# Sniffer
Is a program that sniffes the network
## Usuage
To read a file
       ```
       ./main --file <file-name>
       ```
Or you can run
        ```
        ./main -f <file-name>
        ```

To read a package from the network
        ```
        ./main --network
        ```
Or you can run
        ```
        ./main -n
        ```
